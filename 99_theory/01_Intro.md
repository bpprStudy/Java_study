# Java 기본 문법
## Java 기본
### 운영체제와 프로그램
- 프로그램 (Program)
    - 컴퓨터에서 실행될 때 특정 작업을 수행하는 일련의 명령어들의 집합
- 운영체제 (Operating System, OS)
    - 시스템 소프트 웨어

### 컴퓨터의 자료표현
- 비트 (Bit)
- 바이트 (Byte) : 8비트
- 2진수 (Binary)

### 자바 가상 머신 (JVM)
- 자바 바이트코드를 실행할 수 있는 주체
- 모든 바이트고드는 플랫폼에 독립적이며 모든 JVM은 자바 가상 머신 구격에 정의된 대로 자바 바이트코드를 실행
- JRE : 자바 실행 환경
- JDK : 자바 개발 도구

## 조건문
### if - else if - else 문
```java
if(조건식){
    실행할 문장;
} else if {
    실행할 문장;
} else {
    실행할 문장;
}
```

## 반복문
### for 문
```java
for(초기화식; 조건식; 증감식) {
    반복 수행할 문장
}
```

### while 문
```java
while (조건식) {
    반복 수행할 문장;
}
```

### do while 문
```java
do {
    반복 수행할 문장;
} while (조건식);
```
